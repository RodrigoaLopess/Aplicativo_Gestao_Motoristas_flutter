# Projeto Motoristas P2

Aplicativo Flutter simples para cadastro de motoristas com banco local em SQLite.

## O que o projeto faz

- Mostra um resumo inicial
- Lista os motoristas cadastrados
- Cadastra, edita e exclui motoristas
- Exibe uma tela de detalhes

## Tecnologias utilizadas

- Flutter
- Dart
- SQLite com `sqflite`
- `google_fonts`
- `intl`

## Requisitos atendidos

- Navegação entre pelo menos 3 telas com passagem de parâmetros
- Uso de banco de dados local
- Uso de componentes com estado
- Uso de `Drawer` e `BottomNavigationBar`
- Uso de biblioteca do `pub.dev`: `google_fonts`
- Componentes separados em arquivos com parâmetros

## Como executar

```bash
cd C:\Android\Projeto_P2\projeto_motoristas_p2
flutter pub get
flutter run
```
